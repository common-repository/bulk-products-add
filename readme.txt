If you need to add multiple products this plugin will speed up products creation.

== Description ==
When you need to add several new products to the woocommerce shop manually it could become a quite time-consuming task. This plugin helps you to save time and effort and speed up the process of setting up your products. Just specify names of your products (one per line), some basic description, set product visibility, and stock SKU pattern, and hit "Add Bulk" button.

== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress 4.4+
* WooCommerce 3.0+

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WooCommerce products bulk adder, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Product bulk add” and click Search Plugins. Once you’ve found our WooCommerce bulk adder plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently Asked Questions ==

Yes, this plugin works fine with WordPress 4.9, It is also compatible for older WordPress versions upto 5.6.

This plugin is compatible with the latest version of WooCommerce.
